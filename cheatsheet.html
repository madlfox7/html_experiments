Basic HTML Tags

  • <!DOCTYPE html> - Declares the document type and version.
  • <html> - The root element of the HTML page.
  • <head> - Contains metadata and links to external files.
  • <body> - Contains the main content of the page.
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1>Heading</h1>
        <p>This is a paragraph.</p>
    </body>
</html>

Text Elements

  • <h1> to <h6> - Headings, from largest to smallest.
  • <p> - Paragraph.
  • <br> - Line break.
  • <strong> - Bold text.
  • <em> - Italic text.
<h1>Main Heading</h1>
<p>This is a paragraph with a <strong>strong</strong> and <em>italic</em> text.</p>

Links and Images

  • <a href="URL"> - Hyperlink.
  • <img src="image.jpg" alt="description"> - Image element.
<a href="https://example.com">Visit Example</a>
<img src="image.jpg" alt="A description of the image">

Lists

  • <ul> - Unordered list.
  • <ol> - Ordered list.
  • <li> - List item.
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
<ol>
    <li>First</li>
    <li>Second</li>
</ol>

Forms

  • <form> - The form element.
  • <input> - Input fields.
  • <textarea> - Multi-line input fields.
  • <button> - Submit button.
<form>
    <label for="name">Name</label>
    <input type="text" id="name">
    <button type="submit">Submit</button>
</form>

Tables

  • <table> - Table element.
  • <tr> - Table row.
  • <td> - Table cell.
  • <th> - Table header cell.
<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>
