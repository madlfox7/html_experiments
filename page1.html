Short Intro to HTML

  • HTML (HyperText Markup Language) is the standard language used to create web pages.
  • HTML was invented by Tim Berners-Lee.
  • It uses a system of elements and tags to structure content, such as headings, paragraphs, links, and images.
  • HTML works alongside CSS for styling and JavaScript for interactivity.
  • It is the foundation of all websites on the internet and describes the structure of a webpage.

HTML Element

An HTML element consists of an opening tag, content, and a closing tag
HTML can also have empty tags without content and closing tag(for example <hr>, <br>, etc) 

                    <opening_tag>Content</closing_tag>
                
                    <p>This is a paragraph</p>

Note:

  • HTML is not case-sensitive, but it's best practice to write tags in lowercase.
  • Always close your HTML tags properly to avoid unexpected behavior.
  • Use semantic elements like <header>, <nav>, and <footer> for better structure.
  • Opening Tag: The starting tag that defines the element.
  • Content: The information inside the element.
  • Closing Tag: The ending tag that marks the end of the element.

HTML Code Example

HTML code looks like this:

                    
                        <!DOCTYPE html>
                        <html>
                        <head>
                            <title>Sample Page</title>
                        </head>
                        <body>
                            <h1>Hello, World!</h1>
                        </body>
                        </html>
